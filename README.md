# 🎬 Netflix Content Dashboard

An interactive data dashboard built using the Netflix Movies & TV Shows dataset from Kaggle.

## 📊 Key Highlights
- 8,803 total shows analysed
- 19 rating categories
- 515 genre combinations
- 4,527 unique directors
- Content spanning from 1925 to 2021
- 69.6% Movies vs 30.4% TV Shows

## 🛠️ Tools Used
- Power BI / Tableau
- Dataset: [Netflix Dataset on Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
